# Formulaire d'Inscription au Catalogue de Fonctionnalités

Ce projet est une application React qui présente un formulaire d'inscription pour un catalogue de fonctionnalités. Le formulaire inclut des validations complètes et une interface utilisateur réactive.

## Fonctionnalités

- Formulaire d'inscription complet avec validation
- Champs pour les informations personnelles et professionnelles
- Sélection d'intérêts multiples
- Confirmation de soumission réussie
- Design responsive

## Installation

1. Clonez ce dépôt
2. Installez les dépendances:

```
npm install
```

3. Démarrez l'application:

```
npm start
```

L'application sera accessible à l'adresse [http://localhost:3000](http://localhost:3000).

## Structure du Projet

- `src/App.jsx` - Composant principal de l'application
- `src/components/RegistrationForm.jsx` - Composant du formulaire d'inscription
- `src/components/RegistrationForm.css` - Styles du formulaire

## Technologies Utilisées

- React 18
- JavaScript ES6+
- CSS3
